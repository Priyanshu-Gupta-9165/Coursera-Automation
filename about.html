üìù About This Extension

Coursera Automation is a Chrome extension that automates repetitive course interactions on Coursera. It programmatically detects video players, reading pages, and navigation elements to automatically complete lectures, scroll readings, and advance course modules. All processing runs locally in the browser using client-side scripts, ensuring zero data collection and full user privacy while improving learning workflow efficiency.

This extension was designed and developed by Priyanshu Gupta as a productivity tool to reduce repetitive manual actions during online learning. The goal was to create a lightweight and simple automation system that improves course navigation without affecting user control.
