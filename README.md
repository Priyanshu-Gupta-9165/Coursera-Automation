## üí° Pro Tips

- **Speed Demon**: Use the speed control to watch videos at **2x** or **4x** speed. The extension ensures the setting sticks even if Coursera tries to reset it.
- **Background Run**: You can keep the Coursera tab open in the background; the extension will continue to work (ensure the browser doesn't put the tab to sleep).
- **Hard Refresh**: If something looks stuck, a simple page refresh (F5) usually fixes the automation flow instantly.

## üõ† Troubleshooting

**Issue: Automation stopped working**
- **Fix**: Refresh the page. If that fails, click the extension icon, stop, and start again.

**Issue: "Next" button not clicking**
- **Fix**: Ensure the video has fully completed. Some courses have strict timers.

**Issue: Extension icon is missing or gray**
- **Fix**: Go to `chrome://extensions`, toggle Developer Mode off and on, then reload the extension.
